# Examples Directory

This directory contains example scripts and demonstration code for learning how to use various components of the Wipsie system.

## Contents:

- `in_flight_demo.py` - Demonstrates SQS message lifecycle and in-flight status
- `message_lifecycle_demo.py` - Shows how messages flow through the system
- `process_messages.py` - Example message processing script
- `send_sqs_message.py` - Basic SQS message sending example
- `send_sqs_message_fixed.py` - Improved SQS message sending with error handling
- `simple_sqs_test.py` - Simple SQS connectivity test

## Usage:

```bash
# Test SQS connectivity
python examples/simple_sqs_test.py

# Send a test message
python examples/send_sqs_message.py

# Watch message lifecycle
python examples/message_lifecycle_demo.py
```

## Note:
These examples are great for learning and testing individual components!
