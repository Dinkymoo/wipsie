# Archive Directory

This directory contains old, deprecated, or backup files that are kept for reference but are no longer actively used in the project.

## Contents:

- `celery_worker_backup.py` - Backup of the original monolithic Celery worker before refactoring
- `test_celery.py` - Old Celery test file
- `simple_worker.py` - Simple worker implementation for testing
- `start_server.py` - Legacy server startup script
- `test_message.json` - Test message data

## Note:
These files can be safely deleted once you're confident the new modular architecture is working correctly.
