## Data Flow

1. **Copy Event** â†’ Content Script detects copy
2. **Message** â†’ Content Script sends to Background Worker
3. **Storage** â†’ Background Worker saves to Chrome Storage
4. **Display** â†’ Popup reads from storage and shows history
5. **Interaction** â†’ User clicks item to copy back to clipboard
