See the [browser-extension/README.md](browser-extension/README.md) for detailed installation and usage instructions.

### Quick Start

1. Open Microsoft Edge
2. Go to `edge://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `browser-extension` folder
5. Start copying text - your history will be automatically saved!
